Open discrepancy: Reading Cadre

Formula discrepancies found by comparing prior work to the reference workbook

These four items changed this calculator's math on 2026-07-22, based directly on the uploaded reference file — not on assumption:

1. No separate Pre-K bucket. The reference workbook has no "Regular Pre-K" category. Pre-K students are already inside "Basic/Intense/Complex — Pre-K-12," which all explicitly span Pre-K through 12. A prior version of this tool (built from a different, GPT-sourced "New Rule Updates" note) added a standalone Pre-K unit category — that does not appear in the authoritative workbook and has been removed.

2. Instructional Supports is a flat 20% catch-all, not individually modeled positions. The reference computes Instructional Supports as 20% of Division I positions earned — a single line that stands in for Nursing, Driver's Ed, Mental Health, Related Services, Academic Excellence, etc. A prior version of this tool modeled School Nurse and Driver Ed Teacher separately using older Title 14 statute formulas. Those are removed in favor of the reference's single 20% line.

3. Assistant Superintendent applies equally to charters and districts. A prior GPT-sourced update said charters get 0 Assistant Superintendent positions. The reference workbook's actual formula applies identically regardless of LEA type, computed per school first (based on that school's own Division I units plus Principal, AP, Instructional Supports, and Admin Support Professional positions combined — not Division I units alone) and then summed to the LEA. That per-school-then-summed method is what this calculator now uses, and it is not mathematically identical to computing once from the LEA-wide total.

4. 11-Month Supervisor and Transportation Supervisor use continuous division with no floor, ever. An earlier "fix" to this calculator (made in response to a code review that argued from statute text) changed these to floor at/above their first threshold. The reference workbook does no such thing — it is plain continuous division (positions ÷ 150, enrollment ÷ 7,500) at every level, with no IF/FLOOR guard. That earlier "fix" is reverted; the reference is the higher-authority source now that it's available.

Plan forward

  1. Resolve Reading Cadre. Reading Cadre is not a column in the reference workbook's Data sheet — it's computed at the top level as a flat 1-per-LEA at $96,155, the same as before. The conflict with the FY2026 appropriation (which excludes charters and prices the position differently) is still open. See the Register tab.
  2. The three items the reference workbook itself flags as open: Equalization, Local Funding Reform, and Tiering LI/MLL/Vocational weights. These are noted directly in the source file (Calculator!S2:S5) as unresolved policy questions, not something this tool can answer — they need DDOE/legislative input.
  3. Verify the Central Office formulas against actual statute, not just against this workbook. The reference workbook is the best available source of the current calculation logic, but "what the spreadsheet currently computes" and "what's legally correct" aren't guaranteed to be the same thing — that's the entire premise of this project. Assistant Superintendent's per-school-then-summed methodology, in particular, is worth an explicit legal check.
  4. Resolved: the Statewide Superintendent count is 43, not 44. This was flagged as an open discrepancy in an earlier version of this calculator; it turned out to be a bug in this tool, not a real conflict. Delaware has 19 traditional school districts and 24 charter schools (43 independent LEAs). Dover Air Force Base appears as a 44th row in the reference workbook's Data sheet for enrollment/funding attribution, but it isn't an independent LEA — its own Assistant Superintendent and Director fields are 0, and it doesn't get its own Central Office. An earlier version of this calculator's LEA-count logic classified anything not ending in "District" as a charter, which silently swept DAFB into the charter bucket and inflated the total to 44. Fixed 2026-07-22.
  5. Confirm the CTE/Vocational Division I chain independently. Vocational Division I and Vocational Deduct come pre-computed from the reference workbook's Data sheet (itself derived from a per-school CTE participation percentage table). This calculator uses those numbers as given rather than re-deriving them, since re-deriving them independently risks introducing a new error rather than removing one.
  6. Get real formulas for Principal, Assistant Principal, and Admin Support Professionals. The What-If Model tab currently uses step-functions pattern-matched against the 259 real school rows, not a published rule — I could not find or derive an exact formula for these from the reference workbook (they're precomputed values there too). If DDOE or the reference workbook's author can share the actual formula, the What-If tool should be updated to use it and its "Pattern-derived" tags removed.
  7. Food Services Supervisor doesn't appear to be enrollment-driven at all. Only 8 of 259 real schools have a nonzero value, and it's always exactly 1, not scaled to size — it looks like a fixed designation (a specific kitchen/site), not a formula. The What-If tool treats it as a manual yes/no toggle rather than guessing at a threshold.
